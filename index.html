About

Hello, my name is Tynon. I am a full stack software developer with skills in JavaScript, React.js, and Node.js to name a few. Previously to becoming a software developer, I was a software support specialist within the healthcare, financial and software industries where I provided excellent client and internal communication. I like to think of myself as the oil that kept things running smoothly! Also during my time in software support, I was exposed to awesome feedback clients returned about products and some not so awesome feedback. This interested me in coding.

I love creating applications and features that make clients sing along with the skills to improve existing applications to better fit client expectations. It is more difficult than the client facing side but the power you feel being able to think of an idea then create it is amazing. I will admit my passion is pretty broad, I love programming to get an application to complete a task that I find difficult or tedious.
